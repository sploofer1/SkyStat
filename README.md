## Установка
1. Скачай [последний релиз](https://github.com/sploofer1/SkyPlus/releases)
2. Распакуй архив
3. Перейди в Chrome по адресу `chrome://extensions/`
4. Включи «Режим разработчика»
5. Нажми «Загрузить распакованное» и выбери папку с расширением

---

## Возможности

### Статистика
*   **Индикаторы:** плашки по каждому предмету с точным подсчётом долгов.
*   **Английский:** отображение Уроков AI, Тестов, Drilling, STT и SR. Уровень рассчитывается по формуле `current/total`, а не по стандартному цвету API.
*   **Математика:** отображение последней оценки и статуса «автомата» с кастомными emoji.
*   **Специальные предметы:** мониторинг уроков и курсовых.
*   **Управление:** скрытие лишних плашек через настройки.

### Google Calendar
*   Интеграция в правую панель интерфейса.
*   Отображение событий на 7 дней (интервал от -2 до +4 дней от текущей даты).
*   Анимация переключения дней.
*   Просмотр деталей события с активными ссылками при клике.
*   Индикатор прогресса текущего занятия.
*   Автоматическое обновление токена доступа.

### Кастомизация
*   **Шрифт:** выбор из стандартного списка, подключение через Google Fonts URL или загрузка локальных файлов (.ttf, .otf, .woff, .woff2).
*   **Цвета:** индивидуальная настройка акцентов для интерфейса и календаря.
*   **Статусы:** выбор цветов для индикаторов ✅, ⚠️ и ❌.
*   **Фон страницы:** использование сплошного цвета, градиента с любым количеством цветов и углом наклона, изображений или GIF (лимит до 15MB).
*   **Emoji:** установка собственных значков для статусов и оценок по математике (5, 4, 3, 2, автомат).

### Настройки интерфейса
*   Переключатели для управления видимостью элементов: плашки, новости, кружки, точки, журнал, ссылки.
*   Выборочное скрытие плашек для конкретных предметов.
