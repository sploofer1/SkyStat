# SkyPlus

**Браузерное расширение которое превращает Skyeng/Skypro из учебной платформы в удобный рабочий инструмент.**

## Что это

Skyeng и Skypro - хорошие платформы, но в них есть вещи которые мешают работать: 
информация о прогрессе разбросана по страницам, интерфейс нельзя настроить под себя, 
постоянно отвлекают новости и анимации. SkyPlus решает это.

## Возможности

**Статистика**

Плашки с состоянием по каждому предмету появляются прямо на главной странице - 
не нужно заходить в каждый предмет отдельно. Зелёный цвет означает что всё сдано, 
жёлтый - есть предупреждения, красный - долг с количеством.

Для математики отдельная логика: показывается последняя оценка и текущий автомат 
с драматичными индикаторами в зависимости от результата.

Данные берутся напрямую из API и обновляются автоматически каждые 5 минут 
без открытия лишних вкладок.

**Кастомизация**

- Выбор шрифта для всего интерфейса платформы
- Отключение новостного блока на главной
- Скрытие оригинальных кружочков-индикаторов
- Отключение фонового фонарика за курсором

## Установка

1. Скачай архив из раздела [Releases](https://github.com/sploofer1/SkyStat/releases)
2. Распакуй его
3. Открой `chrome://extensions/`
4. Включи "Режим разработчика" в правом верхнем углу
5. Нажми "Загрузить распакованное расширение" и выбери распакованную папку
