# SkyStat
Стильное расширение для мониторинга личной статистики и кастомизации интерфейса Skyeng/Skypro.

## Основные функции
* **Темная тема**: Современный интерфейс в тёмном стиле.
* **Мониторинг прогресса**: Отслеживание долгов по урокам, домашкам и тестам прямо на карточках предметов.
* **Скрытие новостей**: Возможность убрать блок новостей с главной страницы для минимализма.
* **Кастомизация**: Выбор шрифтов (Inter, Comfortaa и др.) для комфортного чтения.
* **Быстрое сканирование**: Автоматический сбор данных по всем предметам за 30-60 секунд.
* Если у вас "**съезжает**" статистика, уменьшите масштаб страницы

## Установка
1. Скачайте репозиторий, и распакуйте его.
2. Откройте `chrome://extensions/`.
3. Включите "**Режим разработчика**" в правом верхнем углу.
4. Перетащите **zip** архив drag-n-drop на страницу.

<p align="center"><i>ИЛИ</i></p>

1. Распакуйте архив, нажмите "**Загрузить распакованное расширение**" и выберите папку.
